# BioQuantix

An AI-powered bio-pharma M&A intelligence terminal for tracking clinical milestones, pipeline gaps, cash runway pressure, shadow signals, and acquisition targets.
[Live site](https://pharma-hunter-web.vercel.app)

## What This Project Covers

- Bio-pharma M&A target intelligence
- Clinical milestone and pipeline monitoring
- Healthcare investment research dashboards

## Tech Stack

- React
- Vite
- JavaScript
- Vercel
